HTML ELEMENTS 

  <h1>
  • The h1 tag can be used to create header text
  • This can be used to create emphasis and draw attention to a certain piece of text
  • The h1 tag is a block level element 
  <p>
  • The p tag can be used to create paragraph text
  • This can be used to create larger blocks of text that don't require specific attributes (such as a img or header tag)
  • Paragraph is a block level element, taking up a set amount of space
  • Different elements can be nested within the paragraph 
 <img>
  • The img tag is used to imbed a specific image into an html document
  • The img tag can pull image from a local harddrive or from a specific source on the internet
  • The img tag is a inline element 
  <ol>
  • The ol tag is used to create ordered lists within an html document
  • The ol tag is a block level elelemtn
  • The ol tag is combined with the <li> tag to create list elements
  • The ol tag is similar to the ul tag, the latter of which uses general bullet points

HTML CHARACTER ENTITIES 

 <
  • The < entity is referred to as an ending tag
  • HTML uses this symbol to show when a html element is done being declared 
 &
  • The & entity is used to denote a special character so it can be written as plain text 
 "
  • The " element is interpreted as the beginning and end of an attribute's value.
  • For example, you would use the " element when denoting a specific style of an element 
     < ul style = "font-size: 30pt" >
    >

CSS SELECTORS 

 .class
  • The CSS class selector matches elements based on the contents of their class attribute.
  • You can assign a class to an html element by typing class = xclass while defining the element in the opening tags 
 :nth-child
  • The nth-child selector can be used to ascribe specific attributes to specified html child elements
  • For example, using the nth-child selector you can have alternating colors for list elements 
 #id
  • The id tag can be used to ascribe a specific css element to an html element marked with a specific id 
 element
  • The element selector is used to ascribe a certain attribute to all html elements
  • You can ascribe attributes with css by declaring p { x elements } in the css file

CSS PROPERTIES 

 background-color
  • The background-color element is used to change the background color of an element
  • The background-color element only takes one parameter, which is the name of a color, or the hex value 
 color
  • The color element is used to change the color of text within an html document
  • For example you can use the color attribute to change the color of text 
 font-size
  • The font-size element is used to change the size of words in an html document
  • For example, you can change the size of text using the font element
